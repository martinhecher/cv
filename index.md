## Projects

### DURAARK

DURAARK is a project funded by the European Union to support the long-term archival of architectural data. We provide a microservice-based system for the processing and long-term archival of architectural data via a REST-API and a graphical web application. See the [project web page](http://duraark.eu) for more information, the [demo page](http://workbench.duraark.eu) for the web application and our [Github page](https://github.com/duraark/duraark-system) for the source code.

I was leading the work package responsible for integrating the software prototypes delivered by our six project partners into a coherent web application and REST-API.

### V-MANIP

V-MANIP is a project funded by the European Space Agency (ESA). It developed a multi-dimensional web application for earth observation data collected by ESA satellites. See the [demo page](http://demo.v-manip.eox.at) for the resulting web application and the [Github page](https://github.com/v-manip/WebClient-Framework) for the source code.

I was designing and developing the application framework, together with two 3D viewer components (terrain viewer, globe viewer).
